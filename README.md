# SHORT-GUN

A simple and fast **URL Shortener API** built using **Node.js**, **Express**, and **MongoDB**.  
It allows you to create and retrieve shortened URLs with unique short IDs.

---

## Features
- Create short URLs using an API endpoint  
- Retrieve stored URLs from the database  
- MongoDB integration via Mongoose  
- Environment-based configuration with dotenv  
- Modular route-based architecture  

---

## Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Environment Management:** dotenv  
- **Utilities:** short-id  
- **Dev Tools:** nodemon
